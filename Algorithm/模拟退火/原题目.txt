3167. 星星还是树

在二维平面上有 n 个点，第 i 个点的坐标为 (xi,yi)。
请你找出一个点，使得该点到这 n 个点的距离之和最小。
该点可以选择在平面中的任意位置，甚至与这 n 个点的位置重合。

输入格式
第一行包含一个整数 n。
接下来 n 行，每行包含两个整数 xi,yi，表示其中一个点的位置坐标。

输出格式
输出最小距离和，答案四舍五入取整。

数据范围
1≤n≤100,
0≤xi,yi≤10000
输入样例：
4
0 0
0 10000
10000 10000
10000 0
输出样例：
28284
难度：中等
时/空限制：1s / 64MB
总通过数：256
总尝试数：482
来源：POJ2420，acwing3167
算法标签
模拟退火
